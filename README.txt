                        ======================================================
                        =================       Projet C      ================
                        =================  Image vectorielle  ================
                        ======================================================

Réalisé par Kiroshan SIVAKUMAR, Lyna ARROUL

Ce projet a pour but de réaliser un programme imitant un logiciel de dessin.
Pour cela, notre projet est divisé en 4 fichiers source avec chacun son header associés et un fichier main.c.

Pour faire fonctionner le programme, il faudra lancer ce fichier main.c.
Il réalisera une boucle infinie permettant de lancer le programme tant que l'utilisateur ne souhaite pas quitter.
Nous pouvons créer des dessins avec 5 formes prédéfinies :
    - Point ;
    - Ligne ;
    - Carré ;
    - Rectangle ;
    - Cercle ;
    - Polygone.

Pour chacune de ces formes un type structure du même nom a été créé afin de stocker les données nécessaires au tracé.
Pour créer une forme, il suffit de saisir la commande associée à la forme. En tapant 'help', il sera possible de voir
les différentes commandes possibles.
